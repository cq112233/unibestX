采用 Vue 3 组合式 API、Vite 5 极速构建以及 UnoCSS 原子化引擎，为您带来现代化的前端开发工作流。

专为 uni-app X 打造，我们深度重构并修复了原版组件中由于原生平台限制导致的大量兼容性问题，如 up-sticky、up-box 等。
